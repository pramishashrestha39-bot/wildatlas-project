# WildAtlas – Wildlife Website

## Author
Pramisha Shrestha

## Project Description
WildAtlas is a responsive wildlife-themed website developed as part of the course CT70A9111 – Software Development Skills: Front-End.

The website provides information about different animals including lions, eagles, rhinos, elephants, turtles, and whales. It demonstrates the use of HTML, CSS, JavaScript, responsive design, Flexbox, CSS Grid, image slideshows, navigation menus, and interactive FAQ sections.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git
- GitHub Pages

## Features
- Responsive navigation menu
- Multi-page website structure
- Animal information pages
- Image slideshow
- FAQ section
- Responsive layout using Flexbox and CSS Grid
- GitHub Pages deployment

## How to Run the Project

### Option 1: Online
Open the deployed website using the GitHub Pages link.

### Option 2: Local
1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in a web browser.
